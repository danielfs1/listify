# listify

## Downloading dependecies
----------------------
### Client
- `cd client`
- run `bower install`

### Server
- `cd server`
- run `npm install`
- install mongodb`

## Running
---------
### Client
- `cd client`
- run `grunt serve`
- close the window that opens

### Server
- `sudo mongod`
- cd into 'server'
- run 'npm test'
- open localhost:3000 in a browser
