# listify

## Downloading dependencies

### Client
- `cd client`
- `bower install`

### Server
- `cd server`
- `npm install`
- install mongodb

## Running

### Client
- `cd client`
- `grunt serve`
- close the window that opens

### Server
- `sudo mongod`
- `cd server`
- `npm test`
- open localhost:3000 in a browser
